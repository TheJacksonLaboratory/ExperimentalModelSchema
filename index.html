Welcome to the documentation for the experimental-model-schema!

The goal of the experimental-model-schema (EMS) is to define a model for the description of genetic, genomic, experimental, and phenotypic information about an experiment with a model organism. // The schema as well as source code in Java, C++, and Python is available from the experimental-model-schema GitHub repository.
